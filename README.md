C++ Zaawansowane Mechanizmy Obiektowe

Repozytorium zawiera zbiór programów i ćwiczeń realizowanych w ramach pracy własnej oraz przygotowań do egzaminu końcowego z przedmiotu Podstawy Programowania w C/C++. 

Kody źródłowe dokumentują praktyczne opanowanie zaawansowanych mechanizmów języka, ze szczególnym uwzględnieniem paradygmatu obiektowego, zarządzania pamięcią oraz programowania generycznego.

📂 Zawartość repozytorium

1. Podstawy Klas i Zarządzanie Cyklem Życia
Klasy_Podstawy.cpp - Listy inicjalizacyjne, delegowanie konstruktorów, idiom RAII oraz składowe statyczne.

2. Polimorfizm i Dziedziczenie
Polimorfizm.cpp - Klasy abstrakcyjne, metody czysto wirtualne oraz polimorfizm dynamiczny.
Zaawansowana_Obiektowosc.cpp`** - Wirtualne destruktory, słowa kluczowe `override` i `final`, kompozycja (relacja "has-a"), implementacja interfejsów oraz bezpieczne rzutowanie `dynamic_cast`.

3. Przeciążanie Operatorów
Operatory.cpp - Przeciążanie operatorów binarnych arytmetycznych (+, -), operatorów porównania (==, !=) z zachowaniem logiki oraz operatora strumieniowego `<<` poprzez funkcje zaprzyjaźnione.

4. Szablony i Bezpieczeństwo (Wyjątki)
Szablony.cpp - Szablony klas, oddzielanie deklaracji od definicji, parametry non-type oraz pełna specjalizacja szablonu.
Wyjatki_Integracja.cpp - Bezpieczne kontenery (`SafeArray`), specyfikator `noexcept` oraz kontrola dostępu do pamięci.
Szablony_Wyjatki.cpp - Szablony funkcji, hierarchia własnych klas wyjątków opartych na `std::runtime_error` oraz kaskadowa obsługa błędów.

Repozytorium stanowi potwierdzenie 95 godzin pracy własnej przewidzianych w karcie przedmiotu.*
